# Política de Privacidade — Scan Fit IA

Última atualização: 25/02/2026

O Scan Fit IA respeita sua privacidade. Esta política explica quais dados são processados e como são usados.

## 1. Dados coletados

- **Peso informado pelo usuário** (digitado ou identificado por OCR).
- **Meta de peso** configurada no aplicativo.
- **Histórico local de pesos** (últimos registros salvos).

## 2. Como os dados são usados

- Exibir histórico e progresso de meta.
- Melhorar a experiência dentro do próprio app.

## 3. Armazenamento

- Os dados do MVP atual são armazenados localmente no dispositivo via armazenamento local (`localStorage`).
- Não há envio automático desses dados para servidores nesta versão.

## 4. Câmera

- A câmera é usada apenas para capturar imagem e tentar identificar o peso via OCR.
- O uso da câmera ocorre somente com permissão do usuário.

## 5. Compartilhamento de dados

- O app não vende nem compartilha dados pessoais com terceiros nesta versão.

## 6. Segurança

- São adotadas práticas básicas para reduzir riscos, porém nenhum sistema é 100% inviolável.

## 7. Direitos do usuário

- Você pode limpar os dados locais removendo os dados do app nas configurações do dispositivo.

## 8. Contato

Para dúvidas sobre privacidade, entre em contato com o responsável do app antes da publicação oficial na loja.
