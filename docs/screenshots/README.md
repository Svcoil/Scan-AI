# Guia de Screenshots (Play Store)

Este guia padroniza imagens de loja para melhorar conversão.

## Formato recomendado
- Celular: 1080 x 1920 (9:16)
- PNG ou JPEG
- Sem excesso de texto pequeno

## Sequência sugerida (5 imagens)
1. **Registro em segundos**
   - Mostrar tela Home + chamada curta.
2. **Leitura por câmera (OCR)**
   - Mostrar tela de captura com foco no escaneamento.
3. **Meta e progresso**
   - Mostrar barra de progresso e status da meta.
4. **Histórico de pesos**
   - Mostrar chips/lista de histórico recente.
5. **Versão Pro (insights)**
   - Mostrar tela paywall e benefício premium.

## Texto curto por imagem (PT-BR)
1. "Registre seu peso em segundos"
2. "Capture com câmera e OCR"
3. "Acompanhe meta e progresso"
4. "Veja seu histórico rapidamente"

## Dicas de qualidade
- Use dados reais ou coerentes nas telas.
- Mantenha contraste alto e legibilidade.
- Evite poluição visual e textos longos.
